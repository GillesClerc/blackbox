  Note : le TMAG5273 est un capteur Hall lineaire I2C (remplace le

  Adresse I2C configurable (defaut 0x35). Pas besoin de GPIO dedie.

  │  GND  ── GND                                        │
  │  SDA  ── I2C SDA (bus partage)                     │
  │  SCL  ── I2C SCL (bus partage)                     │
  │  INT  ── (non connecte, ou GPIO47 si interrupt)    │
  │  A0   ── GND (adresse 0x35)                        │

  └─────────────────────────────────────────────────────┘

  Capteur Hall lineaire 3 axes (X, Y, Z) avec sortie I2C.
  Remplace le A3144E (digital ON/OFF) par une mesure proportionnelle.

  Specifications :
  - Range : ±40 mT (A1) ou ±80 mT (A2) selon variante
  - Resolution : 12-bit par axe
  - Calcul angle integre (CORDIC) : angle X/Y ou X/Z
  - Consommation : ~3 µA en mode veille, ~2.5 mA actif
  - Frequence lecture : jusqu'a 20 kHz
  - VDD : 1.7V - 3.6V (compatible 3.3V directement)

  Usages escape box :
  - Detection distance aimant (chaud/froid, 0-40mm)
  - Detection angle rotation aimant (plateau, cadran)
  - Detection position aimant (X/Y sur une surface)
  - Seuil configurable pour emulation ON/OFF si besoin

                │          │          │
  ┌─────────────┴──────────┴──────────┴─────────────────┐
  │                  ADS7830 (U26)                       │
  │                  (TSSOP-16)                          │
  │                                                     │
  │  +VDD  ── 3.3V + 100nF                             │
  │  REFIN ── 3.3V + 1µF (ref externe ratiometrique)   │
  │  GND   ── GND                                       │
  │  COM   ── GND (borne -IN commune single-ended)     │
  │  SDA   ── I2C SDA (bus partage, via JST-SH)        │
  │  SCL   ── I2C SCL (bus partage, via JST-SH)        │
  │  A0    ── GND ┐                                     │
  │  A1    ── GND ┘ → adresse 0x48                     │
  │                                                     │
  │  CH0 ←── SL1 curseur (fader 1)   ┐                 │
  │  CH1 ←── SL2 curseur (fader 2)   │ potentiometres  │
  │  CH2 ←── SL3 curseur (fader 3)   │ 10kΩ alimentes  │
  │  CH3 ←── SL4 curseur (fader 4)   │ par le meme     │
  │  CH4 ←── RV1 curseur (pot rot 1) │ rail 3.3V que   │
  │  CH5 ←── RV2 curseur (pot rot 2) │ REFIN → mesure  │
  │  CH6 ←── RV3 curseur (pot rot 3) │ ratiometrique   │
  │  CH7 ←── RV4 curseur (pot rot 4) ┘                 │
  │                                                     │
  └─────────────────────────────────────────────────────┘

  ADC SAR 8 bits, 8 entrees single-ended, pilote par I2C. Remplace
  les 3 entrees ADC ESP32 (GPIO1-3 liberes → toggles SW1/SW2 + reserve).

  Mode ratiometrique : REFIN cable sur le rail 3.3V qui alimente les
  potentiometres → la lecture est independante des variations du rail.
  Cote firmware : PD1PD0=01 dans le command byte (ref interne 2.5V OFF).

  ⚠ Mapping canaux C2-C0 du command byte NON sequentiel — voir
  docs/datasheets/ADS7830.md (table commandes CH0..CH7 dans l'ordre :
  0x84, 0xC4, 0x94, 0xD4, 0xA4, 0xE4, 0xB4, 0xF4).

  Protocole : write 1 byte commande → read 1 byte resultat.
  Pas de registre de configuration, pas de WHO_AM_I (probe = ACK).
