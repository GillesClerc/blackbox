  ┌─────────────────────────────────────────────────────────────────────────┐
  │               PCB SATELLITE CAPTEURS (~30×50mm)                       │
  └─────────────────────────────────────────────────────────────────────────┘

  Tous les capteurs suivants sont sur un seul petit PCB, connecte au
  bus I2C via JST-SH 4 pins. Chaque capteur a son propre decouplage.

  Detection touche : lecture registre status (polling ou interrupt).

  │                                                        │
  │  ANT1 ── antenne PCB ──┐ antenne boucle integree      │
  │  ANT2 ── antenne PCB ──┘ sur le PCB main ou FPC ext.  │
  │                                                        │
  └────────────────────────────────────────────────────────┘

  Note : le PN532 est sur le PCB main (pas sur satellite) car
  l'antenne NFC doit etre pres de la face du cube et peut etre
  realisee directement en cuivre sur le PCB (antenne boucle ~40mm).

  Alternative : module PN532 breakout avec antenne FPC externe.

  Adresse I2C : 0x48 (fixe en I2C mode, pas configurable).
  Protocole : NXP PN532 I2C frame format (pas raw I2C registers).


  ┌─────────────────────────────────────────────────────────────────────────┐
  │               A3144E — Capteur a effet Hall                          │
  └─────────────────────────────────────────────────────────────────────────┘

                     3.3V
                      │
                   ┌──┴──┐
                   │10kΩ │ (pull-up)
                   └──┬──┘
                      │
                      ├──────── GPIO47 (ESP32-S3)
                      │
  ┌───────────────────┴───────────────────────────┐
  │                   A3144E                       │
  │                  (SOT-23)                      │
  │                                                │
  │  VDD  ── 3.3V (range 4.5-24V, mais 3.3V OK   │
  │          avec version A3144E-LH ou equiv.)     │
  │  GND  ── GND                                   │
  │  OUT  ── open drain → pull-up 10kΩ → GPIO47   │
  │                                                │
  └────────────────────────────────────────────────┘

  Sortie open-drain : LOW quand aimant detecte, HIGH sinon.
  Seuil detection : ~10-20 mT (depend du modele exact).
  Aimant neodyme N52 ~5mm a < 10mm de distance.

  Note : verifier tension min VDD du modele exact. Certains A3144
  demandent 4.5V min — dans ce cas, alimenter en 5V et ajouter
  un diviseur de tension ou level shifter vers GPIO47 (3.3V).
