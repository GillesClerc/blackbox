  Le bq24075 gere le power path : le systeme est alimente par USB
  quand disponible, sinon par la batterie. La batterie charge en
  parallele sans impacter l'alimentation systeme (DPPM).

  ┌─────────────────────────────────────────────────────────────────────────┐
  │               SORTIE AUDIO — FILTRE RC + CONNECTEUR                   │
  └─────────────────────────────────────────────────────────────────────────┘

  La sortie du PCM5122 est differentiee mais en mode "line out" classique :

               PCM5122                Filtre RC               Ampli / Jack
            ┌──────────┐          (passe-bas ~72kHz)
            │          │
   OUTL ────┤ Pin 6    ├───┤ 100Ω ├───┬──────────────→ OUTL vers ampli
            │          │              │                 (PAM8406 ou jack)
            │          │           ┌──┴──┐
            │          │           │22nF │
            │          │           │cer. │
            │          │           └──┬──┘
            │          │              │
            │          │             GND
            │          │
   OUTR ────┤ Pin 7    ├───┤ 100Ω ├───┬──────────────→ OUTR vers ampli
            │          │              │
            │          │           ┌──┴──┐
            │          │           │22nF │
            │          │           │cer. │
            │          │           └──┬──┘
            │          │              │
            │          │             GND
            └──────────┘

  Calcul filtre : fc = 1 / (2π × 100Ω × 22nF) ≈ 72 kHz
  Attenuation du bruit delta-sigma HF sans affecter la bande audio (20-20kHz)
