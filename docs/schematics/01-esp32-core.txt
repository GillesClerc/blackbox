  Note : le bouton RESET n'est pas nécessaire en production (USB CDC
  gère le reset via DTR/RTS). Pour le debug, un bouton entre EN et
  GND peut être ajouté sur le PCB.


  ┌─────────────────────────────────────────────────────────────────────────┐
  │               DECOUPLAGE MODULE                                       │
  └─────────────────────────────────────────────────────────────────────────┘

  Placer au plus pres des pins VDD du module :

  - 1× 10µF ceramique (bulk)
  - 2× 100nF ceramique (HF, pres de chaque pin 3V3)
  - Via GND direct sous chaque capa

  Le module WROOM-1 contient deja son propre decouplage interne,
  mais les capas externes sont necessaires pour les transitoires
  SPI/I2S hautes frequences.

  Notes :
  - 5.1kΩ sur CC1/CC2 : detection USB device (5V, max 3A)
  - 22Ω serie sur D+/D- : impedance matching USB 2.0
  - L'ESP32-S3 a un PHY USB interne, pas besoin de transceiver
  - VBUS alimente le bq24075 (charge + power path), pas directement l'ESP32
