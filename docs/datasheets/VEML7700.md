## VEML7700

> _Synthèse générée par component-research. Datasheet PDF : [./VEML7700.pdf](./VEML7700.pdf) (Vishay rev. 1.8, 28-Nov-2024). Application note : [./VEML7700-AN-designing.pdf](./VEML7700-AN-designing.pdf)._

**Fabricant** : Vishay Semiconductors
**Catégorie** : Capteur de lumière ambiante (ALS) numérique I²C, 16 bits, avec canal blanc
**Référence officielle** : VEML7700-TT (tape & reel standard, suffixes commerciaux)
**Datasheet source** : https://www.vishay.com/docs/84286/veml7700.pdf

## Vue d'ensemble

Capteur de lumière ambiante à interface I²C avec photodiode à filtre type C (close-to-eye response, technologie Filtron™). Convertisseur 16 bits intégré, gain et temps d'intégration logiciels, deux canaux de mesure (ALS + white). Plage 0 lx → ~120 klx (jusqu'à ~167 klx avec gain 1/8×). Résolution descendant à 0,0036 lx/count (gain 2×, IT 800 ms). Mode shutdown < 1 µA. Très utilisé pour bracket lux d'écrans, éclairage adaptatif, détection jour/nuit.

## Package & Footprint

- **Package** : OPLGA (Open Light Guide LGA), surface mount, 4 pins
- **Dimensions** : 6,8 × 2,35 × 3,0 mm (L × W × H) — hauteur incluant la lentille

## Interface

- **Type** : I²C, deux fils SCL/SDA
- **Vitesse max** : Fast-mode 400 kHz
- **Adresse I²C** : `0x10` (7 bits, fixe — pas de pin d'adresse)
- **Endianness** : registres 16 bits, **LSB-first** (low byte transmis en premier après le command code)
- **Format trame** : `[ADDR+W] [CMD] [DATA_LO] [DATA_HI]` en écriture ; `[ADDR+W] [CMD] [Sr] [ADDR+R] [DATA_LO] [DATA_HI]` en lecture

## Registres clés

Tous les registres sont 16 bits, accédés via un command code 8 bits.

| Cmd | Nom | Reset | Description | Bits remarquables |
|---|---|---|---|---|
| `0x00` | ALS_CONF | `0x0001` | Configuration ALS | `[12:11]` ALS_GAIN, `[9:6]` ALS_IT, `[5:4]` ALS_PERS, `[1]` ALS_INT_EN, `[0]` ALS_SD |
| `0x01` | ALS_WH | `0x0000` | Seuil haut IRQ (16 bits) | Comparé au registre ALS |
| `0x02` | ALS_WL | `0x0000` | Seuil bas IRQ (16 bits) | Comparé au registre ALS |
| `0x03` | Power Saving | `0x0000` | Mode économie | `[2:1]` PSM (0–3), `[0]` PSM_EN |
| `0x04` | ALS | RO | Données canal ALS (16 bits) | Mises à jour à chaque IT |
| `0x05` | WHITE | RO | Données canal white (16 bits) | Mises à jour à chaque IT |
| `0x06` | ALS_INT | RO | Statut interruption | `[15]` int_th_low, `[14]` int_th_high — lecture clear |

### Décodage ALS_CONF (0x00)

**ALS_GAIN** (bits 12:11) :
| Val | Gain |
|---|---|
| `00` | 1× |
| `01` | 2× |
| `10` | 1/8× |
| `11` | 1/4× |

**ALS_IT** (bits 9:6) — temps d'intégration :
| Val | IT |
|---|---|
| `1100` | 25 ms |
| `1000` | 50 ms |
| `0000` | 100 ms |
| `0001` | 200 ms |
| `0010` | 400 ms |
| `0011` | 800 ms |

**ALS_PERS** (bits 5:4) — nombre d'échantillons consécutifs hors-seuil avant IRQ :
| Val | Persistance |
|---|---|
| `00` | 1 |
| `01` | 2 |
| `10` | 4 |
| `11` | 8 |

**ALS_INT_EN** (bit 1) : 0 = interrupt désactivée, 1 = activée
**ALS_SD** (bit 0) : 0 = mesure active, 1 = shutdown

### Power Saving (0x03)

| Mode (PSM) | Refresh time (IT + sleep) |
|---|---|
| 0 (Mode 1) | IT + 500 ms |
| 1 (Mode 2) | IT + 1000 ms |
| 2 (Mode 3) | IT + 2000 ms |
| 3 (Mode 4) | IT + 4000 ms |

Bit 0 = PSM_EN. PSM réduit la conso moyenne en espaçant les mesures (l'AN détaille la corrélation gain/IT/PSM ↔ courant typique).

### ALS_INT (0x06) — flags d'interruption

- bit 14 = `int_th_high` (seuil haut dépassé)
- bit 15 = `int_th_low` (seuil bas franchi)
- La lecture du registre clear les flags.

### Résolution lx/count (extrait Vishay AN)

Table partielle (cf. application note pour le tableau complet) :

| Gain | IT | lx/count | Plage max (lx) |
|---|---|---|---|
| 2× | 800 ms | 0,0036 | ~236 |
| 1× | 800 ms | 0,0072 | ~472 |
| 1× | 100 ms | 0,0576 | ~3 775 |
| 1/4× | 100 ms | 0,2304 | ~15 100 |
| 1/8× | 25 ms | 1,8432 | ~120 000 |

## Séquence d'initialisation

```
1. Power-on : appliquer VDD (2,5–3,6 V), attendre tPOR ≈ 2,5 ms
2. Écrire ALS_CONF (0x00) avec ALS_SD=1 (chip déjà en shutdown au reset, mais explicite)
3. Configurer gain, IT, persistance, interrupt enable (ALS_SD reste à 1 pendant la conf)
4. Optionnel : écrire ALS_WH (0x01), ALS_WL (0x02), Power Saving (0x03)
5. Réécrire ALS_CONF avec ALS_SD=0 pour activer la mesure
6. Attendre tWAKE ≈ 4 ms (Filtron init) + IT avant de lire ALS (0x04)
7. Boucle de lecture : lire ALS (0x04) à intervalle ≥ refresh_time
```

L'AN insiste : tout changement de gain ou IT requiert un cycle SD=1 → conf → SD=0 + attente IT pour invalider la mesure en cours.

## Exemple ESP-IDF

```c
#include "driver/i2c_master.h"
#include "esp_check.h"
#include "esp_log.h"

#define VEML7700_ADDR        0x10
#define VEML7700_REG_CONF    0x00
#define VEML7700_REG_ALS     0x04
#define VEML7700_REG_WHITE   0x05

#define ALS_GAIN_1X     (0u << 11)
#define ALS_IT_100MS    (0u << 6)
#define ALS_PERS_1      (0u << 4)
#define ALS_INT_DIS     (0u << 1)
#define ALS_SD_ON       (1u << 0)
#define ALS_SD_OFF      (0u << 0)

static const char *TAG = "veml7700";

static esp_err_t veml7700_write16(i2c_master_dev_handle_t dev, uint8_t reg, uint16_t v)
{
    uint8_t buf[3] = { reg, (uint8_t)(v & 0xFF), (uint8_t)(v >> 8) }; // LSB first
    return i2c_master_transmit(dev, buf, sizeof(buf), 100);
}

static esp_err_t veml7700_read16(i2c_master_dev_handle_t dev, uint8_t reg, uint16_t *v)
{
    uint8_t lo_hi[2] = {0};
    ESP_RETURN_ON_ERROR(
        i2c_master_transmit_receive(dev, &reg, 1, lo_hi, 2, 100),
        TAG, "read 0x%02X", reg);
    *v = (uint16_t)lo_hi[0] | ((uint16_t)lo_hi[1] << 8);
    return ESP_OK;
}

esp_err_t veml7700_init(i2c_master_bus_handle_t bus, i2c_master_dev_handle_t *out)
{
    i2c_device_config_t cfg = {
        .dev_addr_length = I2C_ADDR_BIT_LEN_7,
        .device_address  = VEML7700_ADDR,
        .scl_speed_hz    = 400000,
    };
    ESP_RETURN_ON_ERROR(i2c_master_bus_add_device(bus, &cfg, out), TAG, "add_device");

    // Power-on settle
    vTaskDelay(pdMS_TO_TICKS(3));

    // Shutdown pour configurer
    ESP_RETURN_ON_ERROR(veml7700_write16(*out, VEML7700_REG_CONF, ALS_SD_ON), TAG, "sd_on");
    // Gain 1x, IT 100ms, interrupt off, sortir de shutdown
    uint16_t conf = ALS_GAIN_1X | ALS_IT_100MS | ALS_PERS_1 | ALS_INT_DIS | ALS_SD_OFF;
    ESP_RETURN_ON_ERROR(veml7700_write16(*out, VEML7700_REG_CONF, conf), TAG, "conf");
    // tWAKE + IT avant 1ère lecture valide
    vTaskDelay(pdMS_TO_TICKS(4 + 100));
    return ESP_OK;
}

esp_err_t veml7700_read_lux(i2c_master_dev_handle_t dev, float *lux)
{
    uint16_t raw = 0;
    ESP_RETURN_ON_ERROR(veml7700_read16(dev, VEML7700_REG_ALS, &raw), TAG, "als");
    // Résolution gain 1x / IT 100ms = 0.0576 lx/cnt (cf. table Vishay)
    *lux = raw * 0.0576f;
    return ESP_OK;
}
```
